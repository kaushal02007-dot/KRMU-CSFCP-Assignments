**Overview**

This assignment demonstrates practical skills in computer science fundamentals, open-source tools, and career planning. It aims to help students:

Set structured and measurable goals for professional growth.

Explore certifications and skill-building opportunities.

Build and maintain a strong online presence.

Engage in technical events like hackathons or open-source projects.

Create a roadmap for long-term career development aligned with personal aspirations and industry trends.

**Assignment Components**

_**1. Computational Thinking (CO1)**_

**Objective:** Apply computational thinking to design a career-related automation system.

**Tasks:**

Define the problem clearly.

Create a pseudocode algorithm.

Design a flowchart using Draw.io or hand-drawn scan.

Example systems: Job Application Tracker, Study Planner with Task Reminder, Resume Keyword Analyzer.

_**2. Linux & Bash Automation (CO2)**_

**Objective:** Practice Linux commands and automate tasks using Bash scripting.

**Tasks:**

Execute 10 key Linux commands for file management and permissions.

Write a Bash script to automate an academic or project task (e.g., creating a career-folder structure).

Capture screenshots of commands and script execution.

_**3. Emerging Technology Exploration (CO3)**_

**Objective:** Research and summarize an emerging technology domain.

**Tasks:**

Choose a domain (e.g., AI, Cybersecurity, IoT, Cloud, Blockchain).

Create a 1-page infographic covering:

Technology overview

Job roles and salary trends

Relevant certifications

Example of an Indian startup using the technology

_**4. Career Planning & Professional Readiness (CO4)**_

**Step 1:** SMART Goals

Create 3 goals (short, medium, long-term) with measurable indicators.

**Step 2:** Certification Research

Research 2 relevant certifications with details: name, provider, duration, cost, skills, and alignment with SMART goals.

**Step 3:** LinkedIn Profile Update

Add profile picture, summary, education, and at least 3 skills.

Include a screenshot and profile URL in the report.

**Step 4:** Hackathon / Contest / Open-Source Plan

Register or plan participation. Include name, date, preparation steps, and proof.

**Step 5:** Career Roadmap

Develop a 500–800 word roadmap highlighting key milestones, skills, certifications, internships, and year-wise academic/professional growth plan.

_**5. Report Compilation**_

Submit a single PDF or DOCX including:

Algorithm & Flowchart

Linux commands and Bash script screenshots

Infographic

SMART goals and certifications

LinkedIn profile screenshot

Hackathon proof/plan

Career roadmap

**Reflection section (150–250 words)**
